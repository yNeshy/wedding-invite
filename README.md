# Wedding Invitation – Mayssa & Aziz

A minimal, mobile-first static wedding invitation built with Astro and Tailwind CSS.

## Prerequisites

- Node.js 18+

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build for production

```bash
npm run build
```

Static files are output to `dist/`.

## Preview production build

```bash
npm run preview
```
